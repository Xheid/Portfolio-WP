== Description ==

Testimonials is an easy-to-use plugin that allows to add testimonial to the sidebar as a widget or to embed them into a page or post using <code>[testimonials]</code> short code. 


<h4>Plugin Demo : </h4>
<a href="http://oiiio.tech" target="_blank">Click here to see Plugin Demo</a>

<h4>How to use :</h4>
Where you wanna see the testimonials just place this short code there :
<pre>[testimonials]</pre>

<h4>Process bar color :</h4>

This testimonials plugin allow You to change process bar color. You can use any color or HTML hex code.

<h4>Font color :</h4>

This testimonials plugin allow You to change font color. You can use any color or HTML hex code.

<h4>Name color :</h4>

This testimonials plugin allow You to change Name color. You can use any color or HTML hex code.

<h4>Process bar speed :</h4>

This testimonials plugin allow You to Select process bar speed here. Default value is 10000. If you increase value, the scrolling speed will be slower. If you decrease value, scrolling speed will be faster.


== Installation ==

There are a few options for installing and setting up this plugin.

= Upload Manually =

1. Download and unzip the plugin
2. Upload the 'scroll-up-oiiio' folder into the '/wp-content/plugins/' directory
3. Go to the Plugins admin page and activate the plugin

= Install Via Admin Area =

1. In the admin area go to Plugins > Add New and search for "scroll up oiiio"
2. Click install and then click activate
